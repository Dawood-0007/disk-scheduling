# Disk Scheduling Visualizer

Disk scheduling Visualizer with 12 HDD Algorithms and 4 SSD algorithms. 
  
## Code Setup

### Frontend

```
cd frontend
npm install
npm run dev
```

---

### Backend

```
cd backend
npm install
npm run start
```
